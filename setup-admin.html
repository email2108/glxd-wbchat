Thiết lập Admin GLXD

Thiết lập tài khoản admin duy nhất cho hệ thống

Hướng dẫn lấy Telegram Chat ID:

  1. 1. Mở Telegram và tìm bot @glxdshop_bot
  2. 2. Gửi tin nhắn bất kỳ (ví dụ: "hello")
  3. 3. Bot sẽ trả lời với mã OTP
  4. 4. Chat ID của bạn là ID số Telegram của bạn
  5. 5. Hoặc gửi /start cho bot @userinfobot để lấy Chat ID
